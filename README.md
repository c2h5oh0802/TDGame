# 魔王守衛戰：流派覺醒

這是一個單檔 HTML 塔防遊戲。

## 本機執行

- 直接雙擊 `TD.html` 即可遊玩。

## 免費上線（GitHub Pages）

1. 建立 GitHub Repo（例如 `tdgame`）
2. 上傳本資料夾內容（至少要有 `index.html` 與 `TD.html`）
3. 到 Repo 的 `Settings` -> `Pages`
4. Source 選 `Deploy from a branch`
5. Branch 選 `main`，Folder 選 `/ (root)`
6. 儲存後等待 1~3 分鐘，取得公開網址

公開網址格式通常為：

- `https://<你的帳號>.github.io/<你的repo>/`

任何人打開網址就能直接玩。
